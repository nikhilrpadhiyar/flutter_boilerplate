# Changelog

## [1.0.0] - 2024-01-01

### Added
- Initial release
- Clean Architecture with feature-first folder structure
- GetX state management, DI, and routing
- Dio networking with Auth, Error, and Logging interceptors
- GetStorage for local persistence
- Material 3 theming with light/dark modes
- ARB-based localization (EN, HI, ES)
- Firebase Core, Auth, Firestore, Analytics, Crashlytics, Messaging integration
- Typed exception and failure hierarchy
- Shared widget library (AppButton, AppShimmer, AppErrorWidget, AppEmptyWidget)
- Unit tests for validators and models
- GitHub Actions CI/CD (analyze, test, build)
